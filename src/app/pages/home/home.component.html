Welcome. This is a sandbox app used to test and demonstrate various features of our store class. StoreBase provides a way of storing data, usually retrieved from a remote api, and acting on that data (find, add, replace, delete). The class also maintains the state of the store (pending, working, ready, error).

In this app, you will find examples of various use-cases of stores derived from StoreBase. The data for the examples comes from various remote apis and a delay is added to the fetch to exaggerate the time it takes to make a remote api call. This makes it more clear when the app is pulling data from a remote source or using cached data.
